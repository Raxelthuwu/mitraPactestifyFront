# Sistema de Gestión Territorial - Frontend

  Frontend React + TypeScript para gestión territorial.

  ## Requisitos

  - Node.js
  - npm

  ## Instalación

  ```bash
  npm install

  ## Variables de entorno

  Crear .env.local:

  VITE_API_BASE_URL=http://IP_O_DOMINIO_DEL_BACKEND

  El frontend envía el JWT automáticamente en:

  Authorization: Bearer <token>

  ## Desarrollo

  npm run dev

  ## Build

  npm run build
